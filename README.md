# Predictive Maintenance — Manufacturing ML Model

## Problem Statement
Predict machine failure in industrial environments
using sensor data — reducing unplanned downtime.

## Dataset
AI4I 2020 · 10,000 records · 6 failure types

## Results
| Model            | F1-Score | Recall |
|------------------|----------|--------|
| Logistic Reg.    |  0.72    |  0.68  |
| Random Forest    |  0.89    |  0.86  |
| XGBoost          |  0.91    |  0.89  |
